# Set wallpaper on OSX

Agent for update wallpaper on Mac everyday.

## Setup

1. Install [lunchy](https://github.com/eddiezane/lunchy)

    ```sh
    gem install lunchy
    ```
1. Install wallpaper

    ```sh
    git clone git@github.com:sigorilla/wallpaper.git
    cd wallpaper
    ./setup.py
    lunchy start wallpaper
    ```
